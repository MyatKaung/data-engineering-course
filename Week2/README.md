# Week 2 Homework

This directory contains the solution for Week 2 homework.

## Code

The analysis script `analyze_data.py` downloads the Green and Yellow taxi data for the required periods, and calculates the file size and row counts.

You can run it using:
```bash
python3 analyze_data.py
```
(Make sure to use a python environment with `urllib` standard library).
